Come configurare
  1. Vai su developers.facebook.com/apps e crea un'app tipo Business
  2. Aggiungi il prodotto Instagram + collega un account Business/Creator a una Facebook Page
  3. Copia App ID e App Secret nel .env (META_APP_ID, META_APP_SECRET)
  4. Lancia docker compose restart api
  5. Avvia ngrok per esporre localhost: ngrok http 8000 e copia l'URL HTTPS in PUBLIC_BASE_URL
  6. Clicca "Autorizza con Meta" qui sotto per fare l'OAuth flow
